# Ultron v1
AI powered coding assistant with neural code editor capabilities.

## Features
- Full-featured IDE UI with React and Monaco Editor.
- Node.js backend for file management and AI proxying.
- AI Chat integration.
- Voice coding support.

## Project Structure
- `client/`: React frontend.
- `server/`: Node.js express backend.

## Getting Started
1. Install dependencies in both client and server directories.
2. Start the server: `cd server && npm start`
3. Start the client: `cd client && npm run dev`
